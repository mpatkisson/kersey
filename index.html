Kersey ================ Kersey is a flexible and simple engine for importing various data sources into MongoDB using Mongo's batch write capabilities. All writes to MongoDB are safe. The number of writes per batch is configurable as is the form of the BSON document ultimately stored in Mongo. Support or Contact
